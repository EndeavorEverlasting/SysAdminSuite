# SysAdminSuite

A repo for IT solutions

## Scripts

- `Scripts/Get-MonitorInfo.psm1`: Retrieve monitor details such as model, serial number, native resolution, connection type and video adapter.
- `Scripts/Map-Printer.ps1`: Map a network printer given a printer path.
- `Scripts/Take-Screenshot.ps1`: Capture the primary screen and save it as a PNG file.
- `Scripts/Test-Network.ps1`: Perform a simple connectivity test to a host.
- `Scripts/Invoke-FileShare.ps1`: Access a network file share and list its contents.
